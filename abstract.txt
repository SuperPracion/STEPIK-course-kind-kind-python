Полиморфизм:
ad hoc - в python не используется
параметрический - переопределение метод классов для нужд определенного класса
Инициализация и финализация - магические методы отвечающие за создание и удаление ЭК (__init__, __del__)
Магический метод __new__ -

__getattribute__(self, item) - магический метод вызывается при получении свойства класса с именем item
__getattr__(self, item) - магический метод возвращающий значение запрашиваемого аттрибута ЭК

__settatr__(self, key, value) - магический метод реализующий присвоение значения ЭК

__delattr__(self, item) - магический метод удаляющий аттрибут ЭК

__call__ - () магический метод вызова

__str__ - магический метод отображает информацию об объекте  для пользователя (print, str)
__repr__ - магический метод отображает информацию об объекте для отладки
__len__ - магический метод вызываемый при вызову функции получения длинный len()
__abs__ - магический метод вызываемый при вызове функции преобразования по модулю abs()

__add__ - магический метод отвечающий за операцию сложения
__sub__ - магический метод отвечающий за операцию вычитания
__mul__ - магический метод отвечающий за операцию умножения
__truediv__ - магический метод отвечающий за операцию деления

__eq__ - маг. метод отвечающий за операцию равенства ==
__ne__ - маг. метод отвечающий за операцию не равно !=  not (__eq__)
__lt__ - маг. метод отвечающий за операцию меньше <
__le__ - маг. метод отвечающий за операцию меньше или равно <=
__gt__ - маг. метод отвечающий за операцию больше >
__ge__ - маг. метод отвечающий за операцию больше или равно >=
